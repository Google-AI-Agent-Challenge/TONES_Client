[Front-End Development Context]
- Core Web/Framework: React 19.2.4 + Next.js 16.2.6 (최신 App Router 아키텍처 채택)
- Language: TypeScript 5 (엄격한 정적 타입 시스템 적용)
- Styling: TailwindCSS 4 (@tailwindcss/postcss 및 포스트프로세서 연동) + CSS Variables 기반 테마
- Data Layer: 
  * Supabase Client SDK (DB 데이터 직접 쿼리 및 CRUD 처리 - @supabase/supabase-js 2.49.4)
  * Next.js Server Actions (서버 사이드 보안 데이터 통신, API Key 노출 방지)
  * Node-fetch 기반 백엔드 연동 (FastAPI 로컬 서버로의 API 리다이렉트 지원)
- AI & Analytics:
  * @google/generative-ai (Gemini 1.5 Pro 탑재 - @google/generative-ai 0.21.0)
  * 대화형 인공지능 기반 VOC 분석 및 실시간 다차원 감성 진단 엔진 (Positive / Neutral / Negative)
  * 규칙 기반 예외 처리 폴백 기능 (Gemini API 호출 제한 혹은 오류 대응 최적화)
- Icons/UI Elements: Lucide React (프리미엄 팩터 아이콘 세트 - lucide-react 1.16.0)

[Architectural Decisions]
1. Next.js 16 + React 19 최신 릴리스 도입
   - Server Actions를 통한 프론트엔드와 백엔드 간의 불필요한 REST API 보일러플레이트 제거.
   - 서버 사이드 렌더링(SSR) 및 정적 최적화로 초기 로딩 속도 극대화.
2. Direct Supabase Integration & Hybrid Redirect Architecture
   - FastAPI가 활성화되지 않았을 때는 Supabase SDK로 프론트엔드 서버에서 데이터에 직접 쿼리하여 데이터베이스 접근 속도를 단축.
   - NEXT_PUBLIC_FASTAPI_URL 환경 변수가 주어지면, API Fetch 및 Server Action을 통해 FastAPI 백엔드로 자연스럽게 트래픽을 리다이렉트하도록 설계하여 로컬 머신 러닝 엔진 및 외부 인프라 통합 유연성 확보.
3. Gemini AI 기반 VOC 진단 자동화 (App Route Handler)
   - app/api/analyze-review 라우트를 구축하여 리뷰 저장 시 실시간으로 감성 진단(0~1 score), 키워드 추출, 이슈 타입 분기, AI 요약 생성을 트랜잭션 내에서 한 번에 해결.
   - API 장애 대응을 위한 내장 정규식 기반 Rule-based Fallback 엔진 탑재로 가용성 99.9% 보장.
4. 클라이언트 사이드 실시간 속성 분석 및 엑셀(CSV) 내보내기 기능
   - `page.tsx` 내에서 "성분 / 트러블", "제형 / 발림성", "용기 / 디자인"과 같은 핵심 VOC 속성을 추출하기 위한 실시간 스코어 연산 기능(calculateScores) 구현. 감성 점수와 키워드를 기반으로 대시보드 상태를 즉각 동기화.
   - BOM(Byte Order Mark, \uFEFF)을 포함한 CSV 파일 자동 생성을 통해 마이크로소프트 엑셀 등의 뷰어에서 한글 깨짐 현상을 원천 방지하고 신속한 오프라인 데이터 분석 지원.
5. 프리미엄 다크 모드 및 글래스모피즘 UI 아키텍처
   - globals.css에 CSS 변수 기반 다크 테마(#121214)와 일관된 UI 색상 매트릭스를 정의하고, 현대적인 글래스모피즘(Glassmorphism) 및 다차원 카드 UI 레이아웃을 제공.
   - Lucide React 아이콘 및 유연한 애니메이션 컴포넌트와의 결합으로 모던하고 고급스러운 비주얼 선사.
